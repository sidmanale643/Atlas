A brain-shaped index of your life

Memories aren't stored in lists; they're distributed across cortical regions. Neurogram maps each memory to its dominant brain area, creating a spatial index that mirrors how your mind actually organizes experience.

Rotate the atlas. Zoom into regions. Watch memories cluster near the neural pathways that encode them.

03

Connected to your agents

MCP Server

Neurogram runs as an MCP server. Any AI client (Claude Desktop, Cursor, or custom agents) can read, write, and search your memory graph through a standard protocol.

Semantic search

Vector embeddings via Transformers.js run locally. Search by meaning, not keywords. "That conversation about the project deadline" finds the right memory even if those words were never used.

Relationship graph

Entities resolve across memories. The same person, place, or concept links every memory it touches, building a knowledge graph that grows richer with every entry.
